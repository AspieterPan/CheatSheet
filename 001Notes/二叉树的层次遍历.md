# 二叉树的层次遍历

Top :: [[王道 - 数据结构]] - 第五章 - 树与二叉树 - 5.3.2

## 算法思想:
---
1. 初始化一个辅助队列
2. 根结点入队
3. 若队列非空,则队头结点出队,访问该结点,并将其左、右孩子插入队尾 (如果有的话)
4. 重复 3 直至队列为空

## 代码实现
---

```cpp
void Levelorder(BiTree T){
	InitQueue(Q);
	BiTree p;
	EnQueue (Q,T);
	while (!IsEmpty(Q)){
		DeQueue (Q, p);
		visit(p);
		if(p->lchild!=NULL)
			EnQueue(Q,p->lchild);
		if(p->rchild!=NULL)
			EnQueue(Q,p->rchild);
```
