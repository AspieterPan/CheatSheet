# 数据寻址 3 堆栈寻址
Top:: [[Computer Compose]] - 第四章 - 指令系统 - 4.2.4

- 堆栈寻址:
	- 操作数存放在堆栈中,隐含使用堆栈指针 (SP) 作为操作数地址。
	- 堆栈
		- 是存储器或专用寄存器组中一块特定的按 " 后进先出 (LIFO) " 原则管理的存储区,该存储区中被读/写单元的地址是用一个特定的寄存器给出的,该寄存器称为堆栈指针 (SP)。
		- 堆栈可用于函数调用时保存当前函数的相关信息
		- 硬堆栈
			- 寄存器
			- 不需要寻址
		- 软堆栈
			- 主存
			- 需要一次寻址
