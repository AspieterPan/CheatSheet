# 单链表的插入删除

Top :: [[Computer Datastructure]] - 第二章 - 线性表 - 2.3.2.1

## 按位序插入（带头结点）
---

```cpp
bool ListInsert(LinkList &L, int i, ElemType e){
	if(i<1)
		return false;
	LNode *p;
	int j=0;
	p=L;
	while(p!=NULL && j<i-1){
		p=p->next;
		j++;
	}
	if(p==NULL)
		return false;
	LNode *s = (LNode *)malloc(sizeof(LNode));
	s->data = e;
	s->next=p->next;
	p->next=s;
	return true;
}

typedef struct LNode{
	ElemType data;
	struct LNode *next;
}LNode, *LinkList;
```

		return false;
